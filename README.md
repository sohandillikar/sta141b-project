## STA 141B Project

Goal: Find the best apartment complex in Davis based on the following:

- Price (Apartments.com)
- Distance from UC Davis campus
- Distance from a grocery store
- Distance from a bus line
- Number of crimes within a certain radius
- Reviews of the apartment complex
